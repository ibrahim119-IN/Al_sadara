# Hero Video Background

Place your hero background video files here:

## Required Files

1. **hero-background.mp4** - Main video file (MP4 format)
2. **hero-background.webm** (optional) - WebM format for better browser support

## Recommended Specifications

- **Resolution**: 1920x1080 (Full HD) or higher
- **Duration**: 15-30 seconds (will loop)
- **File size**: Keep under 10MB for fast loading
- **Compression**: Use H.264 codec for MP4
- **Audio**: Remove audio track (video is muted)

## Tips for Best Performance

1. Use video compression tools like HandBrake
2. Keep the video short (it loops)
3. Avoid complex transitions
4. Dark/industrial themes work best with the overlay
